About This Project

Current Invention my team is working on.

1. Heat Stroke Preventer for Labourers
Problem Solved: Outdoor workers suffer in summers.
Idea Description: Hat sensor monitoring temp/humidity, alerting for breaks.
Implementation Steps: DHT sensor on Arduino; buzzer; app link to the manager; hat prototype.

2. Sign Language Translator Cap Problem Solved: Communication barriers.
Idea Description: Camera to AI then ear-piece connected wirelessly to our phone.
Implementation Steps: Gesture mapping with a Raspberry Pi camera.

3. Public Transport Overcrowd Alert (For vehicle and humans)
Problem Solved: Unsafe crowding.
Idea Description: Camera in a place where overcrowding generally happens and AI counts the number of people.
Implementation Steps: AI model to count people.
